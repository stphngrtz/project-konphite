== Motivation

Project Konphite is a self-study project which offers ground to try out and learn things. Take this documentation as an example. The idea is not to document Project Konphite or any of its services but to implement a automated documentation pipeline. I've never used Asciidoctor and PlantUML or set up a pipeline to publish to GitHub Pages. Instead of waiting for a great use-case where I could try these things, I've decided to implement some stupid hello world stuff, which allows me to focus on the technical challenges.
